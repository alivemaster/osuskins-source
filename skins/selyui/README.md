## Changelog

### 1.3 - Jan 22, 2025

- Add rainbow approachcircle and set as default(You can change it back to white which is available in `Extras/ApproachCircle`)
- Restore old(1.0, 1.1) hitcircle(remastered) and hitsound(Available in `Extras/HitCircle` and `Extras/Hitsound`)
- New design for the right bottom corner on songselect screen
- New ranking screen design!
- New pause/fail screen background design
- Add support to 3:2 aspect ratio

### 1.21 - Aug 14, 2024 (2ND ANNIVERSARY!)

- Reduce cursor glow
- Some minor fixes

### 1.2 - Dec 10, 2023

- New UI (almost all of the elements are optimized based on former design)
- New hitcircle & cursor style
- New gameplay hitsounds (stolen from multiple skins)
- New animations (recreated with blender)
- Add 21:9 support
- New skinnable sprites support (`songselect-top`, `songselect-bottom`, `rank-forum`)
- Reduce file size by a half (compared to v1.1)

### 1.1 - Feb 24, 2023

- All sprites recreated with Figma (except animations)
- New color scheme (Almost all solid colors replaced with gradients so the osk is 40MB now while 1.0 is less than 10MB lol)
- New cursor style. Add pink cursor
- New grade sprites
- New `menu-background`
- Add transparent normal sized `selection-mode-over` sprite [as LuckInferno said](https://osu.ppy.sh/community/forums/topics/1629078?n=2)
- Remove `sliderstartcircle` and `sliderstartcircleoverlay`

### 1.0 fix2 - Aug 22, 2022

- Fixed some issues. The Filename is same as the first release(`- Selyui -.osk`)

### 1.0 fix1 - Aug 19, 2022

- Fixed some issues. The Filename is same as the first release(`- Selyui -.osk`)

### 1.0 - Aug 14, 2022

- First release
